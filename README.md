# ShopBrowsify Website

ShopBrowsify is a full-stack (MERN) e-commerce project developed by Team IT Elites.

## User Functionality

1. Browse Products:
   - View a list of products with images and prices.

2. Search Products:
   - Search for products by name, price, category, and company name.

3. <img width="960" alt="product_listing" src="https://github.com/jitendrakumararkha/ShopBrowsify-website/assets/101549973/1b76891b-eb5e-453f-beda-81ace96ac009">

   - Interact with the product list.

4. Add to Cart:<img width="956" alt="products in user's cart" src="https://github.com/jitendrakumararkha/ShopBrowsify-website/assets/101549973/1ab306e6-5c60-4930-91f5-b6d274ec2449">

 - Add desired products to the shopping cart.

5. Purchase Products:
   - Complete the purchase process after adding products to the cart.
   - User authentication is required.

## Admin Functionality

1. Add Products:
   - Admins can add new products with details such as name, price, category, and company name.
<img width="957" alt="Adding product" src="https://github.com/jitendrakumararkha/ShopBrowsify-website/assets/101549973/94bcee8d-e85e-492d-a1be-4a3be7fa2e2e">

2. Update Product Details:
   - Modify product characteristics as needed.
<img width="960" alt="updating-product" src="https://github.com/jitendrakumararkha/ShopBrowsify-website/assets/101549973/bfb88edf-95e4-4e3b-9b1c-72791141a7cb">

3. Delete Products:
   - Remove specific products from the current product list.

4. User Authentication:
   - Authenticate users using JWT tokens and a login system.

## Technologies Used

- Backend: Node.js
- Frontend: React.js
- Database: MongoDB
